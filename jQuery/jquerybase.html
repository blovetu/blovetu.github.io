jQuery 操作DOM元素
attr() 控制元素的属性 attr(属性名，属性值) attr()例子 
$("#a1").attr("href" , "www.imooc.com");
var $url = $("#a1").attr("href");
$("#tip").html($url);
作用是设置或者返回元素的属性，其中attr(属性名)格式是获取元素属性名的值,attr(属性名，属性值)格式则是设置元素属性名的值。
html()
text()		 操作元素的内容 $("#content").html();
$("#content").text()		 html()
text()例子 
var $content = "<b>唉，我又变胖了！</b>";
$("#html").html($content);
$("#text").text($content);
使用html()和text()方法操作元素的内容，当两个方法的参数为空时，表示获取该元素的内容，而如果方法中包含参数，则表示将参数值设置为元素内容。
addClass()
css()		 操作元素的样式 $("#content").css();
$("#content").addClass("red")		 addClass()
css()例子 
$("#content").css("background-color","red");
$("#content").addClass("red");
addClass()和css()方法可以方便地操作元素中的样式，addClass()中的参数为增加元素的样式名称，css()直接将样式的属性内容写在括号中。
removeAttr(name)
removeClass(class)		 移除属性和样式 $("a").removeAttr("href");
$("a").removeClass("hide")		 removeAttr()
removeClass例子 
$("#al").removeClass("blue");
$("#a1").removeAttr("href");

使用removeAttr(name)和removeClass(class)分别可以实现移除元素的属性和样式的功能，使用removeAttr(name)方法中参数表示移除属性名，removeClass(class)方法中参数则表示移除的样式名
append() 向元素内追加内容 $("body").append(<div>你好<div>) append()例子

$("body").append(rethtml());
append(content)方法的功能是向指定的元素中追加内容，被追加的content参数，可以是字符、HTML元素标记，还可以是一个返回字符串内容的函数。
appendTo() 向被选元素内插入内容 $(content).appendTo(selector) appendTo()例子 
 var $html = "<span class='red'>小青蛙</span>"
$($html).appendTo("div");
appendTo()方法也可以向指定的元素内插入内容，参数content表示需要插入的内容，参数selector表示被选的元素，即把content内容插入selector元素内，默认是在尾部。
before()
after()		 在元素前后插入内容 $(selector).before(content)
$(selector).after(content)		 before()
after()例子 
var $html = "<span class='red'>兄弟。</span>"
$(".green").after($html);
before()和after()方法可以在元素的前后插入内容，它们分别表示在整个元素的前面和后面插入指定的元素或内容,其中参数content表示插入的内容，该内容可以是元素或HTML字符串。
clone() 复制元素 $(selector).clone() clone()例子 
$("body").append($(".red").clone());
生成一个被选元素的副本，即复制了一个被选元素，包含它的节点、文本和属性,其中参数selector可以是一个元素或HTML内容
replaceWith()

$(content).replaceAll(selector)		 replaceWith()

 替换元素或元素中的内容，但它们调用时，内容和被替换元素所在的位置不同
wrap()
wrapInner()		 包裹元素和内容 $(selector).wrap(wrapper)
$(selector).wrapInner(wrapper)		 wrap()
wrapInner()例子 
$(".red").wrapInner("<em></em>");
$(".red").wrap("<div></div>")
 进行元素的包裹，wrap()用于包裹元素本身，wrapInner()则用于包裹元素中的内容
each() 遍历元素 $(selector).each(function(index)) each()例子 
$("span").each(function (index) {
if (index == 1) {
$(this).attr("class", "red");
} 
});
 遍历指定的元素集合，在遍历时，通过回调函数返回遍历元素的序列号,参数function为遍历时的回调函数，index为遍历元素的序列号，它从0开始。
remove()
empty()		 删除元素 $(selector).each(function(index)) remove()
empty()例子 
$("span").remove(".red");
$("span").empty(this);
 remove()方法删除所选元素本身和子元素，该方法可以通过添加过滤参数指定需要删除的某些元素，而empty()方法则只删除所选元素的子元素。

jQuery 事件与应用
ready() 页面加载时触发 $(document).ready(function(){})
等价于$(function(){});		 ready()例子 
$(document).ready(function(){
$("#btntest").bind("click",function(){
$("#tip").html("我被点击了！");
});
});
ready()事件类似于onLoad()事件，但ready()只要页面的DOM结构加载后便触发，而onLoad()必须在页面全部元素加载成功才触发，ready()可以写多个，按顺序执行。
bind() 绑定元素的事件 $(selector).bind(event,[data] function)
$("#test").bind("change click",function(){});		 bind()例子 
$(document).ready(function(){
$("#btntest").bind("click",function(){
$("#tip").html("我被点击了！");
});
});
为被选元素添加一个或多个事件处理程序，并规定事件发生时运行的函数可绑定多个事件，绑定时事件名用空格隔开，事件名要加引号 多个事件名写在一个引号内
hover() 切换事件 $(selector).hover(over，out); hover()例子 
$(function () {
$("div").hover(
function () {
$(this).addClass("orange");},
function () {
$(this).removeClass("orange")
})
});
当鼠标移到所选元素上时，执行方法中的第一个函数，鼠标移出时，执行方法中的第二个函数
toggle() 绑定多个函数 $(selector).toggle(fun1(),fun2(),funN(),...) toggle()例子 
$("div").toggle(
function(){$(this).html("苹果");},
function(){$(this).html("荔枝");},
function(){$(this).html("香蕉");});
toggle()方法可以在元素的click事件中绑定两个或两个以上的函数，同时，它还可以实现元素的隐藏与显示的切换,注意：toggle()方法支持目前主流稳定的jQuery版本1.8.2，在1.9.0之后的版本是不支持的。
unbind 移除元素绑定的事件 $(selector).unbind(event,fun) unbind()例子 
$(function () {
$("div").bind("click",
function () {
$(this).removeClass("backcolor").addClass("color");
}).bind("dblclick", function () {
$(this).removeClass("color").addClass("backcolor");
})
$("#btntest").bind("click", function () {
$("div").unbind();
$(this).attr("disabled", "true");
});
});
unbind()方法可以移除元素已绑定的事件,其中参数event表示需要移除的事件名称，多个事件名用空格隔开，fun参数为事件执行时调用的函数名称
one 绑定元素的一次性事件 $(selector).one(event,[data],fun) one()例子 
$(function () {
var intI = 0;
$("div").one("click", function () {
intI++;
$(this).css("font-size", intI + "px");
})
});
one()方法可以绑定元素任何有效的事件，但这种方法绑定的事件只会触发一次,参数event为事件名称，data为触发事件时携带的数据，fun为触发该事件时执行的函数。
trigger() 手动触发指定的事件 $(selector).trigger(event) trigger()例子 
$(function () {
$("div").bind("change-color", function () {
$(this).addClass("color");
});
$("div").trigger("change-color");
});
可以直接手动触发元素指定的事件，这些事件可以是元素自带事件，也可以是自定义的事件，总之，该事件必须能执行,其中event参数为需要手动触发的事件名称。
focus()
blur()		 文本框的事件 $("div").focus(function(){})
$("div").blur(function(){})		 focus()
blur()例子 
$(function () {
$("input").bind("focus", function () {
    $("div").html("请输入您的姓名！");
}).bind("blur", function () {
    if ($(this).val().length == 0)
        $("div").html("你的名称不能为空！");
})
});
focus事件在元素获取焦点时触发，如点击文本框时，触发该事件；而blur事件则在元素丢失焦点时触发，如点击除文本框的任何元素，都会触发该事件。
change() 下拉列表框的事件 $("div").bind("change",function(){}) change()例子 
$(function () {
$("#seltest").bind("change", function () {
if ($(this).val() == "苹果")
    $(this).css("background-color", "red");
else
    $(this).css("background-color", "green");
})
});
当一个元素的值发生变化时，将会触发change事件，例如在选择下拉列表框中的选项时，就会触change事件。

jQuery 动画特效
show()
hide()		 显示元素
隐藏元素		 $(selector).hide(speed,[callback])
$(selector).show(speed,[callback])		 show()
hide()例子 
$(function () {
$("h4").bind("click", function () {
if ($("#hidval").val() == 0) {
    $("ul").show();
    $("#hidval").val(1);
} else {
    $("ul").hide();
    $("#hidval").val(0);
}
});
});
show()和hide()方法用于显示或隐藏页面中的元素,参数speed设置隐藏或显示时的速度值，可为“slow”、“fast”或毫秒数值，可选项参数callback为隐藏或显示动作执行完成后调用的函数名
show()
hide()		 动画效果的方法 $(selector).hide(speed,[callback])
$(selector).show(speed,[callback])		 show()
hide()例子 
$(function () {
$("h4").bind("click", function () {
if ($("#hidval").val() == 0) {
    $("ul").show(3000,function(){
        $("#hidval").val(1);
    })
} else {
    $("ul").hide("slow",function(){
        $("#hidval").val(0);
    })
}
})
});
增加“speed”参数可以实现动画效果的显示与隐藏，同时，如果添加了方法的回调函数，它将在显示或隐藏执行成功后被调用。
toggle() 实现动画切换效果 $(selector).toggle(speed,[callback]) toggle()例子 
$(function () {
var $spn = $("#spnTip");

  })
  });
});
toggle()方法以动画的效果显示和隐藏元素
slideUp()
slideDown()		 向上滑动效果
向下滑动效果		 $(selector).slideUp(speed,[callback])
$(selector).slideDown(speed,[callback])		 slideUp()
slideDown()例子 
$(function () {
$("h4").bind("click", function () {
if ($("#hidval").val() == 0) {
    $("ul").slideUp(1000,function() {
        $("#hidval").val(1);
    })
} else {
    $("ul").slideDown(1000,function(){
        $("#hidval").val(0);
    })
}
})
});
slideUp()用于向上滑动元素，slideDown()用于向下滑动元素,其中speed参数为滑动时的速度，单位是毫秒，可选项参数callback为滑动成功后执行的回调函数名。slideDown()仅适用于被隐藏的元素；slideup()则相反
slideToggle() 向上向下滑动切换 $(selector).slideToggle(speed,[callback]) slideToggle()例子 
$(function () {
var $spn = $("#spnTip");
$("h4").bind("click", function () {
  $("ul").slideToggle(1000,function() {
$spn.html() == "向下滑" ? $spn.html("向上滑") : $spn.html("向下滑");})
  })
});
调用该方法时，如果元素已向上滑动，则元素自动向下滑动，反之，则元素自动向上滑动,其中speed参数为动画效果时的速度值，可以为数字，单位为毫秒，也可是“fast”、“slow”字符，可选项参数callback为方法执行成功后回调的函数名称。
fadeIn()
fadeOut()		 淡入效果
淡出效果		 $(selector).fadeIn(speed,[callback])
$(selector).fadeOut(speed,[callback])		 fadeIn()
fadeOut()例子 
$(function () {
$("h4").bind("click", function () {
    if ($("#hidval").val() == 0) {
        $("ul").fadeIn(1000,function() {
            $("#hidval").val(1);})
    } else {
        $("ul").fadeOut(1000,function() {
            $("#hidval").val(0);})

});
fadeIn()淡入隐藏的元素，fadeOut()可以淡出可见的元素参数speed为淡入淡出的速度，callback参数为完成后执行的回调函数名。
fadeTo() 设置淡入淡出的不透明度 $(selector).fadeTo(speed,opacity,[callback]) fadeTo()例子 
$(function () {

switch (index) {
    case 0:$(this).fadeTo(3000,0.2);break;
    case 1:$(this).fadeTo(3000,0.4);break;
    case 2:$(this).fadeTo(3000,0.6);break;};
});
});
将所选择元素的不透明度以淡入淡出的效果调整为指定的值,其中speed参数为效果的速度，opacity参数为指定的不透明值，它的取值范围是0.0~1.0，可选项参数callback为效果完成后，回调的函数名。
animate() 制作简单的动画效果 $(selector).animate({params},speed,[callback]) animate()例子 
$(function () {
$("span").animate({
    width: "80px",height: "80px"
},
3000, function () {
    $("#tip").html("执行完成!");});
});
animate()方法可以创建自定义动画效果，其中，params参数为制作动画效果的CSS属性名与值，speed参数为动画的效果的速度，单位为毫秒，可选项callback参数为动画完成时执行的回调函数名。
animate() 制作移动位置的动画 $(selector).animate({params},speed,[callback]) animate()例子 
$(function () {
$("span").animate({
    left: "+=100px"
}, 3000, function () {
    $(this).animate({
        height: '+=30px',
        width: '+=30px'
    }, 3000, function () {
        $("#tip").html("执行完成!");
    });
});
});
animate()方法不仅可以制作简单渐渐变大的动画效果，而且还能制作移动位置的动画，在移动位置之前，必须将被移元素的“position”属性值设为“absolute”或“relative”，否则，该元素移动不了。
stop() 停止当前所有动画 $(selector).stop([clearQueue],[goToEnd]) stop()例子 
$(function () {
$("span").animate({left: "+=100px"}, 3000, function () {
$(this).animate({height: '+=60px',width: '+=60px'}, 3000, function () {
  $("#tip").html("执行完成!");});
});
$("#btnStop").bind("click", function () {
$("span").stop();
$("#tip").html("执行停止!");});
});
stop()方法的功能是在动画完成之前，停止当前正在执行的动画效果，这些效果包括滑动、淡入淡出和自定义的动画,两个可选项参数clearQueue和goToEnd都是布尔类型值，前者表示是否停止正在执行的动画，后者表示是否完成正在执行的动画，默认为false。
delay() 延时执行动画效果 $(selector).delay(duration) delay()例子 
 $(function () {
$("span").animate({left: "+=100px"}, 3000, function () {
    $(this).animate({height: '+=60px',width: '+=60px'}, 3000, function () {
    $("#tip").html("执行完成!");});
});
$("#btnStop").bind("click", function () {
    $("span").delay(3000);$("#tip").html("正在延时!");});
});
delay()方法的功能是设置一个延时值来推迟动画效果的执行,其中参数duration为延时值，它的单位是毫秒，当超过延时值时，动画继续执行
