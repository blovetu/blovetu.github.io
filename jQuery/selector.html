选择器名称 选择器格式 选择器描述

jQuery 基础选择器
#id 选择器 $("#my_id") 通过一个id号去查找一个元素
element 选择器 $(“element”) 根据元素名查找元素,其中element就是元素的名称
.class 选择器 $(“.class”) 通过元素的类别属性查找元素,其中.class参数表示元素的CSS类别(类选择器)名称
* 选择器 $(“*”) 获取页面中的全部元素
sele1,sele2,seleN选择器 $(“sele1,sele2,seleN”) 精确的选择任意多个指定的元素,其中参数sele1、sele2到seleN为有效选择器，每个选择器之间用“，”号隔开，它们可以是之前提及的各种类型选择器
ance desc层次选择器 $("ance desc") 通过层次选择器，可以快速定位某一层次的一个或多个元素，其中ance desc是使用空格隔开的两个参数。ance参数（ancestor祖先的简写）表示父元素；desc参数（descendant后代的简写）表示后代元素，即包括子元素、孙元素等等
parent > child选择器 $(“parent > child”) 选择子集元素,child参数获取的元素都是parent选择器的子元素，它们之间通过“>”符号来表示一种层次关系。
prev + next选择器 $(“prev + next”) 查找与“prev”元素紧邻的下一个“next”元素,其中参数prev为任何有效的选择器，参数“next”为另外一个有效选择器，它们之间的“+”表示一种上下的层次关系并且只返回唯的一个元素
prev ~ siblings选择器 $(“prev ~ siblings”) 查找prev 元素之后的相邻元素,获取prev 元素后面全部相邻的同辈元素,其中参数prev与siblings两者之间通过“~”符号形成一种层次相邻的关系，表明siblings选择器获取的元素都是prev元素之后的同辈元素
this特殊选择器 $(this) this，表示当前的上下文对象是一个html对象，可以调用html对象所拥有的属性和方法。
$(this),代表的上下文对象是一个jquery的上下文对象，可以调用jQuery的方法和属性值。

jQuery 过滤性选择器
:first过滤选择器 $(“.div:first”) 一组相同标签元素中的第1个元素
:last过滤选择器 $(“.div:last”) 一组相同标签元素中的最后一个元素
:not过滤选择器 $(“input:not(:checked)+p”) 选择所有元素去除不匹配给定的选择器的元素
:eq(index)过滤选择器 $(".arron:eq(2)") 从一组标签元素数组中，灵活选择任意的一个标签元素,其中参数index表示索引号（即：一个整数），它从0开始
:gt(-index)过滤选择器 $(".arron:gt(3)" 选择匹配集合中所有大于给定index(索引值)的元素
:lt(index)过滤选择器 $(".arron:lt(3)" 选择匹配集合中所有索引值小于给定index参数的元素的元素
:even过滤选择器 $("div:even") 选择索引值为偶数的元素，从0开始计数
:odd过滤选择器 $("div:odd") 选择索引值为奇数的元素，从0开始计数
:header过滤选择器 $(":header") 选择所有标题元素，像h1,h2,h3等
:lang(language)过滤选择器 $(":lang(language") 选择指定语言的所有元素
:root $(":root")过滤选择器 选择该文档的根元素
:animated过滤选择器 $(":animated") 选择所有正在执行动画效果的元素
:contains(text)内容选择器 $("li:contains('土豪')") 选择包含指定字符串的全部元素，它通常与其他元素结合使用，获取包含“text”字符串内容的全部元素对象。其中参数text表示页面中的文字
:has(selector)过滤选择器 $("li:has('p')") 获取选择器中包含指定元素名称的全部元素，其中selector参数就是包含的元素名称，是被包含元素
:parent内容选择器 $("a:parent") 一键选择所有含有子元素或者文本的父级元素
:empty内容选择器 $("a:empty") 选择所有没有子元素的元素（包括文本节点）
:hidden可见性筛选选择器 $("p:hidden") 获取全部不可见的元素，这些不可见的元素中包括type属性值为hidden的元素
:visible可见性筛选选择器 $("p:visible") 获取的是全部可见的元素，也就是说，只要不将元素的display属性值设置为“none”，那么，都可以通过该选择器获取。
[attribute='value']属性选择器 $("li[title='水果'] 获取与属性名和属性值完全相同的全部元素，其中[]是专用于属性选择器的括号符，参数attribute表示属性名称，value参数表示属性值。
[attribute*='value']属性选择器 $("li[title*='最']") 获取属性值中包含指定内容的全部元素，其中[]是专用于属性选择器的括号符，参数attribute表示属性名称，value参数表示对应的属性值
[attribute~='value']属性选择器 $("li[name~='a']") 选择指定属性用空格分隔的值中包含一个给定值的元素
[attribute|='value']属性选择器 $("div[name|='-']") 选择指定属性值等于给定字符串或以该字符串为前缀（该字符串后跟一个连字符'-'）的元素
[attribute!='value']属性选择器 $("div[testattr!='true'] 选择不存在指定属性，或者指定的属性值不等于给定值的元素
[attribute^='value']属性选择器 $("div[name^='hello']") 选择指定属性是以给定字符串开始的元素。
:[attribute$='value']属性选择器 $("div[name$='world'] 选择指定属性是以给定值结尾的元素。这个比较是区分大小写的
:[attribute] $("div[class]") 选择所有具有指定属性的元素，该属性可以是任何值
:[attributeFilter1][attibuteFilterN]属性选择器 $('div[class][div]') 选择匹配所有指定的属性筛选器的元素
:first-child子元素过滤选择器 $("li:first-child") 获取每个父元素中返回的首个子元素，它是一个集合，常用多个集合数据的选择处理
:last-child子元素过滤选择器 $("li:last-child") 获取每个父元素中返回的最后一个子元素，它也是一个集合，常用多个集合数据的选择处理。
:only-child子元素过滤选择器 $("div a:only-child") 如果某个元素是其父元素的唯一子元素，那么它就会被选中
:nth-child子元素过滤选择器 $("div a:nth-child(2)") 选择他们所有父元素的第n个子元素
:nth-last-child子元素过滤选择器 $("div a:nth-last-child(2)") 选择所有他们父元素的第n个子元素。计数从最后一个开始到第一个

jQuery表单选择器

:input表单选择器 $("#frmTest :input") 返回全部的表单元素，不仅包括所有 input 标记的表单元素，而且还包括 textarea、select 和 button 标记的表单元素，因此，它选择的表单元素是最广的
:text表单文本选择器 $("#frmTest :text") 获取表单中全部单行的文本输入框元素，单行的文本输入框就像一个不换行的字条工具，使用非常广泛。
:password表单密码选择器 $("#frmTest :password") 获取表单中全部的密码输入文本框元素
:radio单选按钮选择器 $("#frmTest :radio") 获取表单中的全部单选按钮元素
:checkbox复选框选择器 $("#frmTest :checkbox") 快速定位并获取表单中的复选框元素
:submit提交按钮选择器 $("#frmTest :submit") 获取表单中的这个提交按钮和button元素
:image图像域选择器 $("#frmTest :image") 获取属性值为image的全部元素
:reset图像域选择器 $("#frmTest :reset") 匹配所有重置按钮
:button表单按钮选择器 $("#frmTest :button") 获取“type”属性值为“button”的 input 和 button 这两类普通按钮元素。
:file图像域选择器 $("#frmTest :file") 匹配所有文件域
:enabled表单对象属性选择器 $("input:enabled") 选取可用的表单元素
:disabled表单对象属性选择器 $("input:disabled") 选取不可用的表单元素
:checked表单对象属性选择器 $("#frmTest :checked") 一些元素存在选中状态，如复选框、单选按钮元素，选中时“checked”属性值为“checked”，调用:checked可以获取处于选中状态的全部元素
:selected表单对象属性选择器 $("#frmTest :selected") 获取 select 下拉列表框中全部处于选中状态的 option 选项元素

样式篇

.attr() $("input").attr("checked","true") attr()有4个表达 式：
1.attr(传入属性名)：获取属性的值;2.attr(属性名, 属性值)：设置属性的值;3.attr(属性名,函数值)：设置属性的函数值;4.attr(attributes)：给指定元素设置多个属性值，即：{属性名一: “属性值一” , 属性名二: “属性值二” , … … }
.removeAttr()删除方法 $("input").removeAttr('value') removeAttr()为匹配的元素集合中的每个元素中移除一个属性（attribute）
.text() $('p').text( $(".first").text() ) 得到匹配元素集合中每个元素的文本内容结合，包括他们的后代，或设置匹配元素集合中每个元素的文本内容为指定的文本内容。具体有3种用法：
1..text() 得到匹配元素集合中每个元素的合并文本，包括他们的后代;2..text( textString ) 用于设置匹配元素内容的文本;3..text( function(index, text) ) 用来返回设置文本内容的一个函数
.text()结果返回一个字符串，包含所有匹配元素的合并文本
.html() $('p').html( $(".first").html() ) .html()方法
1.html() 不传入值，就是获取集合中第一个匹配元素的HTML内容;2..html( htmlString ) 设置每一个匹配元素的html内容;3..html( function(index, oldhtml) ) 用来返回设置HTML内容的一个函数.htm()方法内部使用的是DOM的innerHTML属性来处理的，所以在设置与获取上需要注意的一个最重要的问题，这个操作是针对整个HTML内容（不仅仅只是文本内容）
.val() $('input').val("修改的字段") .val()方法主要是用于处理表单元素的值，比如 input, select 和 textarea。.val()无参数，获取匹配的元素集合中第一个元素的当前值.val( value )，设置匹配的元素集合中每个元素的值.val( function ) ，一个用来返回设置值的函数
.addClass() $('.box').addClass("red") 通过动态改变类名（class），可以让其修改元素呈现出不同的效果,.addClass()方法不会替换一个样式类名。它只是简单的添加一个样式类名到元素上
.removeClass() $('.box').removeClass("red") 它的作用是从匹配的元素中删除全部或者指定的class, 如果没有样式名作为参数，那么所有的样式类将被移除
.toggleClass() $('.box').toggleClass("red") toggleClass方法动态添加删除Class，一次执行相当于addClass，再次执行相当于removeClass.toggleClass( className, switch )：一个布尔值，用于判断样式是否应该被添加或移除
.css()样式操作 $('.box').css("color","red");
$(".box").css({"color":"red"});		 获取元素放临时的一些数据CSS属性
.data()元素的数据存储 jQuery.data( element, key, value ) //静态接口,存数据
jQuery.data( element, key ) //静态接口,取数据
.data( key, value ) //实例接口,存数据
.data( key ) //实例接口,存数据
存放临时的一些数据，同样的jquery.removeData(element[,name])和removeData([name])是删除数据
