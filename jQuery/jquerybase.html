jQuery 操作DOM元素
attr()方法控制元素的属性 attr(属性名，属性值) attr()例子 
$("#a1").attr("href" , "www.imooc.com");
var $url = $("#a1").attr("href");
$("#tip").html($url);
作用是设置或者返回元素的属性，其中attr(属性名)格式是获取元素属性名的值,attr(属性名，属性值)格式则是设置元素属性名的值。
html和text方法操作元素的内容 $("#content").html();
$("#content").text()		 html()
text()例子 
var $content = "<b>唉，我又变胖了！</b>";
$("#html").html($content);
$("#text").text($content);
使用html()和text()方法操作元素的内容，当两个方法的参数为空时，表示获取该元素的内容，而如果方法中包含参数，则表示将参数值设置为元素内容。
addClass()和css()操作元素的样式 $("#content").css();
$("#content").addClass("red")		 addClass()
css()例子 
$("#content").css("background-color","red");
$("#content").addClass("red");
addClass()和css()方法可以方便地操作元素中的样式，addClass()中的参数为增加元素的样式名称，css()直接将样式的属性内容写在括号中。
removeAttr(name)和removeClass(class)移除属性和样式 $("a").removeAttr("href");
$("a").removeClass("hide")		 removeAttr()
removeClass例子 
$("#al").removeClass("blue");
$("#a1").removeAttr("href");
var $url=$("#al").attr("href");
$("#tip").html($url);
使用removeAttr(name)和removeClass(class)分别可以实现移除元素的属性和样式的功能，使用removeAttr(name)方法中参数表示移除属性名，removeClass(class)方法中参数则表示移除的样式名
append()方法向元素内追加内容 $("body").append(<div>你好<div>) append()例子 
function rethtml() {
var $html = "
<div id='test' title='hi'>我是调用函数创建的</div>"
return $html;
}
$("body").append(rethtml());
append(content)方法的功能是向指定的元素中追加内容，被追加的content参数，可以是字符、HTML元素标记，还可以是一个返回字符串内容的函数。
appendTo()方法向被选元素内插入内容 $(content).appendTo(selector) appendTo()例子 
 var $html = "<span class='red'>小青蛙</span>"
$($html).appendTo("div");
appendTo()方法也可以向指定的元素内插入内容，参数content表示需要插入的内容，参数selector表示被选的元素，即把content内容插入selector元素内，默认是在尾部。
before()和after()在元素前后插入内容 $(selector).before(content)
$(selector).after(content)		 before()
after()例子 
var $html = "<span class='red'>兄弟。</span>"
$(".green").after($html);
before()和after()方法可以在元素的前后插入内容，它们分别表示在整个元素的前面和后面插入指定的元素或内容,其中参数content表示插入的内容，该内容可以是元素或HTML字符串。
clone()方法复制元素 $(selector).clone() clone()例子 
$("body").append($(".red").clone());
生成一个被选元素的副本，即复制了一个被选元素，包含它的节点、文本和属性,其中参数selector可以是一个元素或HTML内容
replaceWith()和replaceAll()替换内容 $(selector).replaceWith(content)
$(content).replaceAll(selector)		 replaceWith()
replaceAll例子 
var $html = "<span class='red' title='hi'>我是土豪</span>";
$($html).replaceAll($(".green"));
$(".green").replaceWith($html);
 替换元素或元素中的内容，但它们调用时，内容和被替换元素所在的位置不同
wrap()和wrapInner()方法包裹元素和内容 $(selector).wrap(wrapper)
$(selector).wrapInner(wrapper)		 wrap()
wrapInner()例子 
$(".red").wrapInner("<em></em>");
$(".red").wrap("<div></div>")
 进行元素的包裹，wrap()用于包裹元素本身，wrapInner()则用于包裹元素中的内容
each()方法遍历元素 $(selector).each(function(index)) each()例子 
$("span").each(function (index) {
if (index == 1) {
$(this).attr("class", "red");
} 
});
 遍历指定的元素集合，在遍历时，通过回调函数返回遍历元素的序列号,参数function为遍历时的回调函数，index为遍历元素的序列号，它从0开始。
remove()和empty()方法删除元素 $(selector).each(function(index)) remove()
empty()例子 
$("span").remove(".red");
$("span").empty(this);
 remove()方法删除所选元素本身和子元素，该方法可以通过添加过滤参数指定需要删除的某些元素，而empty()方法则只删除所选元素的子元素。

jQuery 事件与应用
页面加载时触发ready()事件 $(document).ready(function(){})
等价于$(function(){});		 ready()例子 
$(document).ready(function(){
$("#btntest").bind("click",function(){
$("#tip").html("我被点击了！");
});
});
ready()事件类似于onLoad()事件，但ready()只要页面的DOM结构加载后便触发，而onLoad()必须在页面全部元素加载成功才触发，ready()可以写多个，按顺序执行。
bind()方法绑定元素的事件 $(selector).bind(event,[data] function) bind()例子 
$(document).ready(function(){
$("#btntest").bind("click",function(){
$("#tip").html("我被点击了！");
});
});
为被选元素添加一个或多个事件处理程序，并规定事件发生时运行的函数可绑定多个事件，绑定时事件名用空格隔开，事件名要加引号 多个事件名写在一个引号内
hover()方法切换事件 $(selector).hover(over，out); hover()例子 
$(function () {
$("div").hover(
function () {
$(this).addClass("orange");},
function () {
$(this).removeClass("orange")
})
});
当鼠标移到所选元素上时，执行方法中的第一个函数，鼠标移出时，执行方法中的第二个函数
